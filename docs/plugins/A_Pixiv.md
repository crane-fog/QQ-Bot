# A_Pixiv

## 简介
简易 Pixiv 图片下载插件，支持按作品 ID 下载图片，也支持清理缓存目录。

## 基本信息
- 插件名：`A_Pixiv`
- 类型：`Group`
- 作者：`cojitaZ`
- 文档由AI生成：`是`

## 触发方式
- `pid<作品ID>`：下载并发送图片
- `p_clean`：清理缓存图片目录

## 生效条件
- 需要在 `plugins.ini` 中启用
- 受 `groups.ini` 群启用控制
- 不要求数据库
- 无额外权限限制

## 配置项
- 无插件专属 `self.config` 配置项

## 执行逻辑
- 读取作品详情，判断是否为多图和是否带 `R-18` 标签
- 下载预览图和原图，打包为转发消息发送
- `R-18` 作品会先在群里提示；当前代码默认既会在群里发送转发消息，也会额外私聊触发者
- `p_clean` 会删除并重建缓存目录

## 外部依赖
- Pixiv 相关远程接口
- 本地目录：`plugins/A_Pixiv/image/`
- 图片处理依赖：`Pillow`

## 注意事项
- 下载逻辑包含缓存与重试
- 插件内置对 `R-18` 内容的特殊发送分支
- `p_clean` 会直接删除并重建 `plugins/A_Pixiv/image/` 下的缓存目录
